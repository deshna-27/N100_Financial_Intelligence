# N100 Financial Intelligence Platform

## Overview

This project builds a financial intelligence platform for Nifty 100 companies.

The pipeline performs:

- Data Loading
- Data Normalization
- Data Validation
- SQLite Database Creation
- Financial Ratio Engine
- CAGR Engine
- Cash Flow KPI Engine
- Data Quality Review
- SQL Exploration
